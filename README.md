# Arduino Journey

A collection of small Arduino projects exploring the Arduino ecosystem through practical, hands-on examples.

This repository covers a variety of sensors, actuators, communication interfaces, and embedded programming concepts. Each project focuses on a specific topic while emphasizing clean, readable code and clear documentation.

## Goals

* Learn and explore the Arduino ecosystem.
* Build practical projects using common electronic components and modules.
* Gain hands-on experience with Arduino libraries and APIs.
* Document experiments, implementations, and project outcomes.
* Create a structured reference for future embedded projects.

## Repository Structure

```text
Arduino-Journey/
├── 01-LDR-Light-Control/
├── 02-Ultrasonic-Distance-Warning/
├── 03-...
└── README.md
```

## Each Project Contains

* Source code (`.ino`)
* Project-specific README
* List of required components
* Circuit diagram or wiring image (when applicable)
* Project description and implementation details
